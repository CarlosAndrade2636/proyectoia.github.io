5.2 Algoritmos de detección de la comunidad

Los algoritmos de detección de comunidad se utilizan para evaluar cómo se agrupan o particionan los grupos de nodos, así como su tendencia a fortalecerse o separarse.

Componentes fuertemente conectados

los algoritmos de componentes fuertemente conectados permiten entender el comportamiento de los nodos que conforman un grafo dirigido y poseen un tipo específico de conectividad. Para que se considere un componente fuertemente conectado o conexo este grafo debe poseer nodos que estén totalmente conectados entre sí. Dentro de un grafo dirigido es posible que no exista una conexión fuerte, aunque es posible que exista una o varias vinculaciones. Se busca con estos algoritmos determinar los componentes fuertemente conectados de mayor tamaño.

Casos de uso: cuándo usar el algoritmo de Componentes fuertemente conectados

- Puede usarse para encontrar el conjunto de empresas en las que cada miembro posee acciones directas y / o indirectas en todos los demás miembros.
- puede utilizar para calcular la conectividad de diferentes configuraciones de red al medir el rendimiento de enrutamiento en redes inalámbricas de múltiples tiendas.
- ueden usar para encontrar dichos grupos y sugerir las páginas o juegos que comúnmente les gustan a las personas del grupo que aún no les han gustado esas páginas o juegos.

Historia y explicacion

SCC es uno de los primeros algoritmos de gráficos, y Tarjan describió el primer algoritmo de tiempo lineal en 1972. La descomposición de un gráfico dirigido en sus componentes fuertemente conectados es una aplicación clásica del algoritmo de búsqueda de profundidad primero.

Uso alternativo

A mediados del siglo pasado, se usaba un algoritmo similar para los estudios relacionados con la sociometría. En esa época, este algoritmo servía para calcular el estatus de un individuo en la sociedad. Más adelante, la Washington State University se percató de que este algoritmo podía también ser de gran utilidad para encontrar moléculas de agua entre productos químicos venenosos. Por otra parte, la empresa Moz creó el MozRank que determina la popularidad de los enlaces. Y, por último, el OPR de [Ryte] analiza la popularidad de un sitio web.
